

1) Backup Black Mesa via Steam and copy to USB drive
2) Deinstall Black Mesa & Call of Duty: Ghosts
3) Backup complete drive using CloneZilla (for worst case scenario)
4) Backup linux partition using rsync

sudo mkdir /mnt/linux
sudo mkdir /mnt/backup

sudo mount /dev/sdX /mnt/linux
sudo mount /dev/sdY /mnt/backup

sudo rsync -axHAWXS --numeric-ids --info=progress2 /mnt/linux/ /mnt/backup/

5) Follow luks-install_uefi.txt, a fresh encrypted Ubuntu should reside on its own partition now

6) Boot Ubuntu Live CD

7)
sudo cryptsetup luksOpen /dev/sda6 sda6-dec

sudo mkdir /mnt/backup
sudo mkdir /mnt/linux

8)
sudo mount /dev/mapper/sdX-dec /mnt/linux
sudo mount /dev/sdY /mnt/backup

9) sudo rsync -axHAWXS --numeric-ids --info=progress2 /mnt/backup/ /mnt/linux/

10) restart
